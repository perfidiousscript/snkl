About

SNKL (pronounced cenacle) is an interactive, user modifiable visualization of authors of the 20th century.

Existing book suggestion algorithms are often opaque and sometimes dead wrong

The purpose of SNKL is to provide an avenue for choosing new literature by demonstrating the various connections between writers.

This allows the reader to start from writers that they have previously enjoyed, select the type of connection that interests them, and search from there

Each writer is displayed on a graph of time and relative style.

By clicking on a writer once color coded connections to other writers with be displayed.

Double clicking on a writer will display more information about a writer.

SNKL was written in AngularJS and d3.js by Sam Moss.
